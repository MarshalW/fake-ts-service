# Read me

可在 [MarshalW/fake-ts-service-client](https://github.com/MarshalW/fake-ts-service-client) 使用这个包。

执行：

```bash
# 构建
npm run build
# 测试
npm test

# 确保 git 提交后，设置小版本更新
npm version patch

# 发布当前更新的 npm 包
npm publish
```

另外，实现了一个命令行工具的示例：

```bash
# 全局安装
npm i -G fake-ts-service

# 运行
fake-service
```
